## Teaching model (summary)

| Question | Answer |
|----------|--------|
| What is a dump/reset? | Short pass backward or lateral to a handler behind the disc to keep possession and reset the stall. |
| Purpose of dump-swing? | Reset buys time and a new angle; swing moves the disc across the field so defense shifts and new downfield lanes open. |
| Where should reset set up? | Behind and slightly wide — not flat behind the thrower, not in the cutting lane. ~10–15 yd spacing. |
| When look dump? | Early — often by stall 3–5, not stall 9. Look downfield first, then turn to reset if covered. |
| How should reset cut? | Small setup move to move the mark, then come back showing hands at a clean angle. |
| When swing? | Right after reset catch — before defense fully resets. Swing handler should already be wide. |
| Third handler? | Holds width on the far side; ready before the dump arrives. |
| Cutters during reset/swing? | Hold/clear downfield; do not clog handler triangle. Attack only after the swing reshapes the field. |
| Defenders? | Mark takes away downfield; reset defender trails; swing side is late after quick disc movement. |
| Beginner mistakes | Waiting too long; reset too close or too far; thrower stands still after dump; swing too late; cutters crowd handlers. |
| Teaching progression | Shape → covered look → reset angle → turn to reset → dump → catch/turn → swing width → swing → field opens → cutter attacks → payoff |

## App lesson mapping

- **A** = thrower with disc  
- **B** = dump/reset handler  
- **C** = swing handler  
- **D–G** = cutters holding/clearing until the swing opens a lane  

No score step — lesson ends with successful continuation or “field opens” payoff.
